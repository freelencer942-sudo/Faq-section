# FAQ Accordion

A clean, modern, and interactive Frequently Asked Questions (FAQ) section built with vanilla HTML, CSS, and JavaScript. 

##  Features

* **Default Active State:** The first FAQ item is open by default when the page loads.
* **True Accordion Functionality:** Only one FAQ item can remain open at a time. Opening a new item automatically closes the previously opened one.
* **Smooth Animations:** Utilizes modern CSS Grid (`0fr` to `1fr` transition) for silky smooth, dynamic height animations without the need for JavaScript height calculations.
* **Clean & Structured Code:** Built with semantic HTML, modular CSS, and concise ES6 JavaScript.
* **Fully Responsive:** Adapts seamlessly to different screen sizes.

##  Technologies Used

* **HTML5:** Semantic structure and content.
* **CSS3:** Styling, layout, and smooth transitions.
* **JavaScript:** Event listeners and class toggling logic.
